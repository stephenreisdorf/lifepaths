# Add Rogue career

Add `data/careers/rogue.yaml` so Rogue is selectable in career selection.

## Scope

- Qualification: DEX 6+.
- Assignments (each with its own Survival / Advancement):
  - Thief — Survival INT 6+, Advancement DEX 6+
  - Enforcer — Survival END 6+, Advancement STR 6+
  - Pirate — Survival DEX 6+, Advancement INT 6+
- Fill in service skills, per-assignment skill tables, events, mishaps, and rank table per the Rogue career entry in the core rules.
