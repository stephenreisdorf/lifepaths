# Add Scout career

Add `data/careers/scout.yaml` so Scout is selectable in career selection.

## Scope

- Qualification: INT 5+.
- Assignments (each with its own Survival / Advancement):
  - Courier — Survival END 5+, Advancement EDU 9+
  - Surveyor — Survival END 6+, Advancement INT 8+
  - Explorer — Survival END 7+, Advancement EDU 7+
- Fill in service skills, per-assignment skill tables, events, mishaps, and rank table per the Scout career entry in the core rules.
