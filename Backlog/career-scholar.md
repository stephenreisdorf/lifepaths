# Add Scholar career

Add `data/careers/scholar.yaml` so Scholar is selectable in career selection.

## Scope

- Qualification: INT 6+.
- Assignments (each with its own Survival / Advancement):
  - Field Researcher — Survival END 6+, Advancement INT 6+
  - Scientist — Survival EDU 4+, Advancement INT 8+
  - Physician — Survival EDU 4+, Advancement EDU 8+
- Fill in service skills, per-assignment skill tables, events, mishaps, and rank table per the Scholar career entry in the core rules.
