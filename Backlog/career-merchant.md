# Add Merchant career

Add `data/careers/merchant.yaml` so Merchant is selectable in career selection.

## Scope

- Qualification: INT 4+.
- Assignments (each with its own Survival / Advancement):
  - Merchant Marine — Survival EDU 5+, Advancement INT 7+
  - Free Trader — Survival DEX 6+, Advancement INT 6+
  - Broker — Survival EDU 5+, Advancement INT 7+
- Fill in service skills, per-assignment skill tables, events, mishaps, and rank table per the Merchant career entry in the core rules.
