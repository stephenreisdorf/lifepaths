# Add Agent career

Add `data/careers/agent.yaml` so Agent is selectable in career selection.

## Scope

- Qualification: INT 6+.
- Assignments (each with its own Survival / Advancement):
  - Law Enforcement — Survival END 6+, Advancement INT 6+
  - Intelligence — Survival INT 7+, Advancement INT 5+
  - Corporate — Survival INT 5+, Advancement INT 7+
- Fill in service skills, per-assignment skill tables, events, mishaps, and rank table per the Agent career entry in the core rules.
