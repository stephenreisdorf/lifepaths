# Add Citizen career

Add `data/careers/citizen.yaml` so Citizen is selectable in career selection.

## Scope

- Qualification: EDU 5+.
- Assignments (each with its own Survival / Advancement):
  - Corporate — Survival SOC 6+, Advancement INT 6+
  - Worker — Survival END 4+, Advancement EDU 8+
  - Colonist — Survival INT 7+, Advancement END 5+
- Fill in service skills, per-assignment skill tables, events, mishaps, and rank table per the Citizen career entry in the core rules.
- Interacts with `Backlog/citizens-drifters-basic-training.md` (Citizens use Assignment Skills for basic training).
