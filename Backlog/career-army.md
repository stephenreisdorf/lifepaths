# Add Army career

Add `data/careers/army.yaml` so Army is selectable in career selection.

## Scope

- Qualification: END 5+. DM−2 if aged 30 or more (aging not yet implemented — see `Backlog/aging.md`).
- Assignments (each with its own Survival / Advancement):
  - Support — Survival END 5+, Advancement EDU 7+
  - Infantry — Survival STR 6+, Advancement EDU 6+
  - Cavalry — Survival DEX 7+, Advancement INT 5+
- Fill in service skills, per-assignment skill tables, events, mishaps, and rank table per the Army career entry in the core rules.
- Military career — depends on commission/officer backlog items for full fidelity.
