# Add Drifter career

Add `data/careers/drifter.yaml` so Drifter is selectable in career selection.

## Scope

- Qualification: automatic (no roll). Always available — also the fallback for failed qualification (see `Backlog/draft-and-drifter-fallback.md`).
- Assignments (each with its own Survival / Advancement):
  - Barbarian — Survival END 7+, Advancement STR 7+
  - Wanderer — Survival END 7+, Advancement INT 7+
  - Scavenger — Survival DEX 7+, Advancement END 7+
- Fill in service skills, per-assignment skill tables, events, mishaps, and rank table per the Drifter career entry in the core rules.
- Interacts with `Backlog/citizens-drifters-basic-training.md` (Drifters use Assignment Skills for basic training).
